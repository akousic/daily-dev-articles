## Summary

Models.dev is a comprehensive open-source database of AI model specifications, pricing, and capabilities. There's no single database with information about all the available AI models. We started Models.dev as a community-contributed project to address this.

## Key Takeaways

- We also use it internally in opencode.
- You can access this data through an API.
