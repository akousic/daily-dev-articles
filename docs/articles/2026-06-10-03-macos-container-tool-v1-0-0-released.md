## Summary

container is a tool that you can use to create and run Linux containers as lightweight virtual machines on your Mac. It's written in Swift, and optimized for Apple silicon. The tool consumes and produces OCI-compatible container images, so you can pull and run images from any standard container registry.

## Key Takeaways

- You can push images that you build to those registries as well, and run the images in any other OCI-compatible application.
- container uses the Containerization Swift package for low-level container, image, and process management.
- You need a Mac with Apple silicon to run container.
