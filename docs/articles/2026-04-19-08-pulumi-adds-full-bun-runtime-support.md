## Summary

Pulumi has announced that Bun is now a fully supported runtime for Pulumi, going beyond its previous role as merely a package manager option. With the new release of Pulumi 3.227.0, developers can set runtime: bun in their Pulumi.yaml and have Bun execute their entire infrastructure program, with no Node.js installation required. When Bun first launched in 2022, the pitch was simple: a faster JavaScript runtime built on JavaScriptCore instead of V8.

## Key Takeaways

- Since then, the project has grown substantially in scope and ambition.
- Bun was invented by Jarred Sumner, first previewed in May 2021, and released as version 1.0 in September 2023.
- The runtime distinguishes itself by bundling a package manager, bundler, and test runner into a single binary, all built with the Zig programming language and powered by JavaScriptCore, Apple's WebKit engine.
