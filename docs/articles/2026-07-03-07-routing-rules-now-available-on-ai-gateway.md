## Summary

Vercel AI Gateway now supports routing rules. Routing rules are firewall-style rules that control which models your team can use, applied at the gateway level instead of in your application code. When a model goes down or gets retired, you usually have to ship a code change to move off it.

## Key Takeaways

- With routing rules, you push one rule and every request reroutes instantly.
- There are two types: Rules apply to every request made with your team's AI Gateway credentials.
- You manage them with the Vercel CLI.
