ReactOS "Open-Source Windows" Reaches The Milestone Of Being Able To Run Half-Life ReactOS, the open-source operating system working for binary compatibility with Microsoft Windows computer programs and drivers, has reached the milestone of being able to enjoy the classic game Half-Life running on this open-source platform.
