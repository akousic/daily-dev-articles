## Summary

On April 27, 2026 the Dutch Ministry of the Interior soft-launched code.overheid.nl, a self-hosted Forgejo instance for Dutch government source code. Project manager Boris Van Hoytema said the platform "was born from the requirement that the ministry has to legally publish [its] source code on a place that [it] owns," and that Forgejo was picked over GitLab because it is fully open source and offers all the freedoms needed for digital autonomy. The week before, I quietly moved my own code in the same direction.

## Key Takeaways

- My canonical Git host is now code.jorijn.com, running Forgejo v15 LTS on a single NUC in a hardened setup.
- Some of my repositories already live there; the rest are queued.
- The longer-term plan is to archive my public GitHub repositories once the migration is complete and point each archive at the new home.
