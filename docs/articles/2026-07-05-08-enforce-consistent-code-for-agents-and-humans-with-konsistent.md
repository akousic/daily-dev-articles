## Summary

konsistent is now open source. konsistent is a CLI linter for TypeScript codebases that enforces structural conventions, giving agents and humans the consistent context they need to implement features correctly. Deterministic, fast, and covers structural patterns that TypeScript and ESLint don't model.

## Key Takeaways

- Configure conventions in a project-level konsistent.json file for concerns like: - Do all files matching pattern X export functions Y and Z?
- - Does every folder that has file X also have file Y?
- - Does every class exported from files matching pattern X implement type Y?
