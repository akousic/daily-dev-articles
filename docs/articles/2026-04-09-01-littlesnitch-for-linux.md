## Summary

Once installed, open the user interface by running littlesnitch in a terminal, or go straight to http://localhost:3031/ . You can bookmark that URL, or install it as a Progressive Web App. Any Chromium-based browser supports this natively, and Firefox users can do the same with the Progressive Web Apps extension.

## Key Takeaways

- The connections view is where most of the action is.
- It lists current and past network activity by application, shows you what's being blocked by your rules and blocklists, and tracks data volumes and traffic history.
- Sorting by last activity, data volume, or name, and filtering the list to what's relevant, makes it easy to spot anything unexpected.
