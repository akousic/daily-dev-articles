## Summary

The code.overheid.nl (Dutch) platform is now live. It is the government-wide code platform for publishing and developing open-source software. The platform is fully self-hosted and supports digital sovereignty.

## Key Takeaways

- For now, this is a pilot using Forgejo, an open-source, European, and sovereign alternative to GitHub and GitLab.
- Not all government organisations can use the platform yet.
- Developers are invited to contribute, with the aim of eventually growing it into a shared Git platform for government bodies.
