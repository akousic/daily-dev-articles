## Summary

I kept running into this problem: I clicked a link in Slack, and it opened in Safari. But I was logged into that AWS account in Chrome Profile 3, and the Figma link should just open in the desktop app, not another browser tab. Set it as your default browser, and it catches every link you click.

## Key Takeaways

- Using rules you define, it routes each link exactly where it belongs - or pops up a fast picker right at your cursor so you can choose on the fly.
- - Rules engine: Route URLs by domain, prefix, regex, or source app.
- Send work stuff to your corporate Edge profile and personal stuff to Safari.
