## Summary

RISCBoy is an open-source portable games console, designed from scratch. This includes: - A RISC-V compatible CPU - A raster graphics pipeline and display controller - Other chip infrastructure: busfabric, memory controllers, UART, GPIO etc. - A PCB layout in KiCad It is a Gameboy Advance from a parallel universe where RISC-V existed in 2001.

## Key Takeaways

- A love letter to the handheld consoles from my childhood, and a 3AM drunk text to the technology that powered them.
- The design is written in synthesisable Verilog 2005, and is intended to fit onto an iCE40-HX8k FPGA.
- This is a LUT4-based FPGA with 7680 logic elements, so fitting a 32 bit games console requires a crowbar and some vaseline, or perhaps just careful design.
